Términos de Servicio

Última actualización: 2025-12-18

1. Aceptación de los términos

Al utilizar post-videos, usted acepta estos términos de servicio y las políticas de TikTok.

2. Descripción del servicio

post-videos proporciona herramientas para automatizar la publicación de contenido en TikTok mediante la API oficial.

3. Responsabilidades del usuario

Usted se compromete a:

  • Usar la aplicación solo con contenido original que posea
  • Cumplir con las Community Guidelines de TikTok
  • No usar la aplicación para spam, contenido fraudulento o violaciones de copyright
  • Mantener la seguridad de sus credenciales de acceso

4. Limitación de responsabilidad

post-videos se proporciona "tal cual". No nos hacemos responsables de:

  • Cambios en la API de TikTok que afecten la funcionalidad
  • Acciones tomadas por TikTok respecto a su cuenta
  • Pérdida de datos o interrupciones del servicio

5. Propiedad intelectual

Usted conserva todos los derechos sobre el contenido que suba a través de post-videos.

6. Modificaciones

Podemos actualizar estos términos ocasionalmente. Los cambios se publicarán en esta página.

7. Terminación

Podemos suspender el acceso si viola estos términos o las políticas de TikTok.
