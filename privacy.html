Política de Privacidad

Última actualización: 2025-12-18

1. Información que recopilamos

Esta Política de Privacidad describe cómo la aplicación 'post-videos' recopila únicamente la información necesaria para funcionar con la API de TikTok:

  • Tokens de acceso OAuth 2.0 (proporcionados por TikTok)
  • Información básica del perfil de TikTok (nombre de usuario, ID)
  • Metadatos de videos que usted elige subir

2. Uso de la información

La información se utiliza exclusivamente para:

  • Autenticar con la API de TikTok
  • Subir videos a su cuenta de TikTok
  • Programar publicaciones según su configuración
  • Nunca vendemos, compartimos o comercializamos sus datos

3. Almacenamiento de datos

Los tokens de acceso se almacenan de forma segura en su propia instancia de n8n. No mantenemos servidores centralizados con sus datos.

4. Seguridad

Implementamos medidas de seguridad estándar de la industria para proteger sus datos. Todos los accesos a la API de TikTok utilizan HTTPS.

5. Derechos del usuario

Puede revocar el acceso a su cuenta de TikTok en cualquier momento desde la configuración de TikTok.
